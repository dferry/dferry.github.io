CSCI 3500: Studio 1

Hello, world!

This is an individual assignment.

Welcome to Operating Systems! In this course we will submit assigments by using GitHub. Let's get started with two simple "Hello, world!" programs that demonstrate the difference between doing standard output with the C standard library versus Linux system calls.

In this studio, you will:

  1. Sign up for a GitHub account, if you don't already have one
  2. Create a GitHub repository for this class
  3. Write a "Hello, world!" program using the C standard library
  4. Write a "Hello, world!" program using Linux system calls
  5. Add these programs to your GitHub repository

Please complete the required exercises below, as well as any optional enrichment exercises that you wish to complete.

As you work through these exercises, please record your answers in a text file called exercises.txt. When finished, commit your source code and exercises to your GitHub repository, and email dferry@slu.edu a link to your solution in the repository with the phrase Hello, world! in the subject line.

Make sure that the name of each person who worked on these exercises is listed in the first answer, and make sure you number each of your responses so it is easy to match your responses with each exercise.

Required Exercises

  1. As the answer to the first exercise, list the names of the people who worked together on this studio.

  2. First, go to www.github.com and register, or sign in if you already have an account. As the answer to this exercise, give your GitHub username.

  3. Next, create a directory structure on your local machine that you will use for this class. Open a terminal and create a new directory named OS with the command mkdir. Then create subdirectories called Labs and Studios. Finally, create a an appropriately named subdirectory for this assignment under Studios.

  4. Next, use the GitHub website to create a new repository for this class.
